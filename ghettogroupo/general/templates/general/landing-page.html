GhettoGroupo is a multi-purpose workspace inspired from several existing applications like Google Classroom, Google Forms, Trello Cards and Kahoot. It provides a single platform for groups of all levels to manage their work, conduct surveys and grow their business. It also provides a platform for individuals willing to work on open projects.

Types of Groups

Different types of groups are available, each with its own specific set of features inorder to thouroughly meet the group requirements

User Hierarchy

The user hierarchy functionality gives certain permissions for certain roles which enables ease of group management

Recommendation System

The recommendation system gives the users a chance to join groups and work on open source projects

Quizzes / Surveys

Quizzes and Surveys can be conducted within a group inorder to test and imporove the productivity

Checklists

The checklists are personalized for each user which has all of his assigned and customized tasks

Restful Services

Based on the role of the user and group type, the RESTFUL services differ accordingly
